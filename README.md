# Multiple Linear Regression - Startup Profit Prediction

## Project Overview
This project predicts the profit of a startup using Multiple Linear Regression.

## Dataset
- 50_Startups.xls

## Features
- R&D Spend
- Administration
- Marketing Spend

## Target
- Profit

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Project Structure

```
multiple-linear-regression-startup-profit-prediction
│
├── backend
│   ├── train_model.py
│   ├── model.pkl
│
├── frontend
│   └── app.py
│
├── 50_Startups.xls
├── requirements.txt
└── README.md
```

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run frontend/app.py
```
